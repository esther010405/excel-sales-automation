# Excel Sales Automation

Projet d'automatisation de rapports Excel réalisé avec Python et Pandas.

## Fonctionnalités

- Lecture de fichiers Excel
- Calcul automatique des montants
- Calcul du chiffre d'affaires total
- Identification du produit le plus vendu
- Génération d'un rapport Excel multi-feuilles

## Technologies

- Python
- Pandas
- OpenPyXL
